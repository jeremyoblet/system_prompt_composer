# Typage statique et qualité Python

- Typage statique (obligatoire)
- Utiliser des annotations complètes (typing, collections.abc).
- Préférer Protocol pour les ports.
- Utiliser dataclass(frozen=True) pour les objets immuables lorsque pertinent.
- Conserver des types précis (éviter Any sauf nécessité).
- PEP 8 + style moderne
- Respect strict de PEP 8.
- Docstrings claires (format cohérent).
- Utiliser des fonctionnalités modernes Python (match/case si utile, f-strings, pathlib, enums, exceptions dédiées).
